# Remote Run Docs

This folder keeps Alpine and other remote-machine run documentation out of the
project root while still making it easy to find.

## Files

- `run_manifest.md` describes the project shape, inputs, dependencies, and
  preset commands needed for remote reconstruction runs.
- `transfer_checklist.md` lists what to upload, what to leave behind, and how to
  retrieve generated outputs.

Related Slurm templates live in `../../remote_jobs/`.
